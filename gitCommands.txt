All the operations below is done from the terminal of any IDE.

1. Clone the repository from github to local:
git clone [link to repo on github]

2. After cloning the repo, rmb to cd into the folder.
3. Make some changes on the project at local.
4. Check the changes made:
git status

5. Add all the changes:
git add .   OR    git add [specific file name]

6. Check the status again:
git status

7. Commit changes to the github repo(still happening in local):
git commit -m [Commit Title] -m [Commit Content]

8. go https://www.youtube.com/watch?v=RGOj5yH7evk&ab_channel=freeCodeCamp.org
for more info on SSH keys for pushing all the commits to github
